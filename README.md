# TrustShield Tech

E-commerce Fake Product Reviews Monitor and Deletion System

## Description
This project detects fake product reviews using a simple AI-based approach.

## Features
- Detect fake reviews
- Identify suspicious reviews
- Simple web interface

## Technology Used
- Python
- Flask

## How to Run
1. Install Flask
2. Run: python app.py
3. Open: http://127.0.0.1:5000
